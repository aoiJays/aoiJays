门控循环单元GRU

并不是每个细节都值得关注

随着喂入序列的变长，序列开头的影响占比会变小

但很有可能序列开头存在重要的关键词

因此我们希望我们的网络能够对序列的不同部分，有所侧重、关注、选择

  • 存储：早期重要信息
  • 跳过：无用信息（网页文章的html代码）
  • 重置：书的章节之间的逻辑中断

门

Rt=σ(XtWxr+Ht−1Whr+br)Zt=σ(XtWxz+Ht−1Whz+bz)R_t = \sigma( X_tW_{xr} +H_{t-1}W_{hr} + b_r ) \\ Z_t = \sigma( X_tW_{xz} +H_{t-1}W_{hz} + b_z ) Rt​=σ(Xt​Wxr​+Ht−1​Whr​+br​)Zt​=σ(Xt​Wxz​+Ht−1​Whz​+bz​)

  • 利用sigmoid函数，全连接层通过输入、隐状态，预测出门的值（0-1之间）

候选隐状态 <- 重置门

正常情况下，隐状态的计算：

Ht=ϕ(XtWxh+Ht−1Whh+bh)H_t =\phi(X_tW_{xh} + H_{t-1}W_{hh} + b_h) Ht​=ϕ(Xt​Wxh​+Ht−1​Whh​+bh​)

我们希望引入RtR_tRt​，对状态进行重置：

  • 设定激活函数为tanh⁡\tanhtanh，确保候选隐状态值在(-1,1)内
  • 使用Hadamard积（矩阵元素对应相乘）：Rt⊙Ht−1R_t \odot H_{t-1}Rt​⊙Ht−1​
    • Rt→0R_t \to 0Rt​→0，此时HtH_tHt​只由当前输入决定。相当于重置了隐状态为一开始的默认值，从头开始
    • Rt→1R_t \to 1Rt​→1，正常的循环神经网络，隐状态照常保存

综上，定义候选隐状态为：

Ht~=tanh⁡(XtWxh+(Rt⊙Ht−1)Whh+bh)\tilde{H_t} =\tanh(X_tW_{xh} + (R_t \odot H_{t-1})W_{hh} + b_h) Ht​~​=tanh(Xt​Wxh​+(Rt​⊙Ht−1​)Whh​+bh​)

隐状态 <- 更新门

上文中，我们计算得到的是候选隐状态

但是如果当前的文本并不让我们感到有意义，我们需要跳过这部分

也就是基本不会修改，直接沿用之前的隐状态

反之，我们希望将当前值更新为最新的隐状态

引入更新门：

Ht=Zt⊙Ht−1+(1−Zt)⊙Ht~H_t = Z_t\odot H_{t- 1} + (1-Z_t)\odot \tilde{H_t} Ht​=Zt​⊙Ht−1​+(1−Zt​)⊙Ht​~​

  • Zt→0Z_t \to 0Zt​→0，相当于完全使用当前新的隐状态
  • Zt→1Z_t \to 1Zt​→1，直接沿用之前的隐状态

总结：

  • 重置门：有助于捕获序列的短期依赖关系
  • 更新门：有助于捕获序列的长期依赖关系

长短期记忆网络LSTM

设计上比GRU更加复杂，但是早了20年

门

  • 输入门：

It=σ(XtWxi+Ht1Whi+bi)I_t = \sigma(X_tW_{xi} + H_{t_1}W_{hi} + b_i) It​=σ(Xt​Wxi​+Ht1​​Whi​+bi​)

  • 遗忘门：

Ft=σ(XtWxf+Ht1Whf+bf)F_t = \sigma(X_tW_{xf} + H_{t_1}W_{hf} + b_f) Ft​=σ(Xt​Wxf​+Ht1​​Whf​+bf​)

  • 输出门：

Ot=σ(XtWxo+Ht1Who+bo)O_t = \sigma(X_tW_{xo} + H_{t_1}W_{ho} + b_o) Ot​=σ(Xt​Wxo​+Ht1​​Who​+bo​)

通过sigmoid激活函数，三个门的值都在(0,1)内

候选记忆元

使用tanh⁡\tanhtanh做激活函数，取值[-1,1]

Ct~=tanh⁡(XtWxc+Ht−1Whc+bc)\tilde{C_t} = \tanh(X_tW_{xc}+H_{t-1}W_{hc}+b_c) Ct​~​=tanh(Xt​Wxc​+Ht−1​Whc​+bc​)

记忆元

记忆元主要来自两个部分：

  • 过去的记忆：即Ct−1C_{t-1}Ct−1​，由遗忘门FtF_tFt​控制保留多少过去的记忆
  • 新的记忆：即当前输入带来的候选记忆元Ct~\tilde{C_t}Ct​~​，由输入门控制引入多少

则：

Ct=Ft⊙Ct−1+It⊙Ct~C_t = F_t\odot C_{t-1}+I_t\odot \tilde{C_t} Ct​=Ft​⊙Ct−1​+It​⊙Ct​~​

这种机制有助于模型记录下非常久远以前的记忆

某种程度上缓解了梯度消失，捕获长距离依赖关系

隐状态

Ht=Ot⊙tanh⁡(Ct)H_t = O_t\odot\tanh(C_t) Ht​=Ot​⊙tanh(Ct​)

  • 确保隐状态仍然在[−1,1][-1,1][−1,1]
  • 输出门接近1：完整保留记忆作为隐状态
  • 输出门接近0：隐状态被重置，只保留了记忆信息

双向RNN

普通的RNN只能考虑到上文，无法考虑到下文：

将隐状态分为正向隐状态、反向隐状态

对于正向隐状态：

Ht→=ϕ(XtWxhfront+H→t−1Whhfront+bhfront)\overrightarrow{H_t}= \phi( X_tW_{xh}^{\text{front}} + \overrightarrow{H}_{t-1}W_{hh}^{\text{front}}+b_h^{\text{front}}) Ht​​=ϕ(Xt​Wxhfront​+Ht−1​Whhfront​+bhfront​)

  • 由输入、上一个前向隐状态得到

对于反向隐状态：

Ht←=ϕ(XtWxhback+H←t+1Whhback+bhback)\overleftarrow{H_t}= \phi( X_tW_{xh}^{\text{back}} + \overleftarrow{H}_{t+1}W_{hh}^{\text{back}}+b_h^{\text{back}}) Ht​​=ϕ(Xt​Wxhback​+Ht+1​Whhback​+bhback​)

  • 由输入、后一个反向隐状态得到

对于输出，我们把Ht→,Ht←\overrightarrow{H_t},\overleftarrow{H_t}Ht​​,Ht​​合并成HtH_tHt​（矩阵连起来）

Ot=HTWhq+bqO_t = H_TW_{hq} + b_q Ot​=HT​Whq​+bq​
